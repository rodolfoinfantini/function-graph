# function-graph

# [Click here to try it!](https://sirvorak.github.io/function-graph/)
It generates the graph of the given math function. Using a self-made graph *"engine"*.

## Operations you can use:
-   sqrt()
-   tan()
-   sin()
-   cos()
-   abs()
-   factorial()
-   pow()
-   pi
-   +-*/^

## Examples:
- sqrt(x)
- x^2
- pow(x,2)
- 2^x
- pi^x
- sin(2x)
- abs(0.1x)
- factorial(x)
- -2x^2+3x+2
- sqrt(cos(x+2)^sin(x^2))
